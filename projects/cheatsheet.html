Ruby Enumberables
Arrays
all? [{|x| ... }] Returns true if block never returns “false”. Default block { |x| x }.
any? [{|x| ... }] Returns true if block ever returns “true”. Default block { |x| x }. Can use any? alone for a boolean.
concat() Add to end of array
delete_if{|x|...} Deletes if statement
delete() or delete_at(index) Deletes if statement
detect { |x| ... } Returns first element where block returns “true”: (1..10).detect {|i| i % 5 == 0} = 2
drop(ind) Returns array with n index gone
each_cons(n) { |x| ... } Splits array into consecutive elements
each_slice(n) { |x| ... } Splits array for every n elements: [1,2,3,4].each_slice(2).to_a = [1,2][3,4]
find_all { |x| ... } Returns array of all elements where block returns “true”.
find_index() Returns index of the first element where block returns “true”: [1,2,3,4].index(3)=2
flatten Make nested array flat. Flatted(1) is to flatten by 1 level. Converts has to an array.
grep(pattern) [{ |x| ... }] Returns array of block return values
include?(value) Return boolean if values is in array.
insert(ind,value) Inserts the given values before the element with the given index
join("") Turn array into string. Within brackets is the serparation.
map { |x| ... } Returns an array with the results from block.
map(&:to_i) Convert array to integers
one? [{ |x| ... }] Returns true if block returns “true” exactly once. Default block { |x| x }. Can use one? alone for a boolean.
pop Removes the last element from self and returns it
reject { |x| ... } Returns array of all elements where block returns “false”.
shuffle Shuffles the array
Sorting
sort { |a, b| ... } Returns array sorted by <=> operator or by block. Default block { |a, b| a <=> b }.
sort_by { |x| ... } string.sort_by {|word| word.length}
max { |a, b| ... } Returns maximum element. Default block { |a, b| a <=> b }.
min { |a, b| ... } Returns minimum element. Default block { |a, b| a <=> b }.
minmax { |a, b| ... } Returns [min, max]. Default block { |a, b| a <=> b }.
Strings
Str.center(space,values) "hello".center(20)= " hello " or "hello".center(20, '123')= "1231231hello12312312"
count Gives the lenght
include?(str) Gives boolean if contains string
chop Removes last letter of string
delete"" Deletes letters: "hello".delete "e","h" = “llo"
each_char {|x|...} Iterate through a string
gsub() Replace characters in a string. 'hello'.gsub(/ [eo]/, 'e' => 3, 'o' => '*') #=> "h3ll*"
Hashes
has_key?(k) Returns boolean if key exists
has_value?(v) Returns boolean if value exists
keys Puts all keys in an array
values Puts all values in an array
invert Switches key/values. Repeats are deleted and last one stays.
merge() Merge 2 hashes
select {|k,v| ...} Returns all that match: h.select {|k,v| v < 200}
SQL Syntax
SELECT is the clause you use every time you want to query information from a database.
WHERE is a popular command that lets you filter the results of the query based on conditions that you specify.
LIKE and BETWEEN are special operators that can be used in a WHERE clause
AND and OR are special operators that you can use with WHERE to filter the query on two or more conditions.
ORDER BY lets you sort the results of the query in either ascending or descending order.
LIMIT lets you specify the maximum number of rows that the query will return. This is especially important in large tables that have thousands or even millions of rows.
Primary Key is a column that serves a unique identifier for row in the table. Values in this column must be unique and cannot be NULL.
Foreign Key is a column that contains the primary key to another table in the database. It is used to identify a particular row in the referenced table.
AS is a keyword in SQL that allows you to rename a column or table in the result set using an alias.
CSS Positioning
position places elements on screen, e.g. absolute
float stacks elements horizontally in a particular
top, left, right, bottom specifies the offsets used in absolute, fixed, and relative positions, e.g. top:10px;left:10px
display sets how the element is placed in the doc flow, e.g. block, inline, none
z-index sets the stacking order of elements, e.g. z- index of 1 is below z-index of 2
overflow sets what happens to content outside
